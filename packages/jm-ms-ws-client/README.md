# jm-ms-ws-client

websocket plugin for jm-ms-core

```javascript

```

- connection

- close
a session closed.

- message

## client events

- connect
start connect to server.

- open
successful connected to server.

- close
disconnected from server.

- error
get an error.

- message
recevied a message.
